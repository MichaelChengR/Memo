# 基于Vue.js的备忘录

## 项目描述
```
通过获取用户输入的数据，实现备忘录的功能
```

### 项目内容
```
用户在输入框内输入待办事项之后，添加到备忘录列表中，可以对备忘录已存在的事项进行修改和删除功能
```
